# Como definir um atalho personalizado

Além do atalho fixo **⌃⌥Espaço**, você pode definir o seu próprio atalho global para mostrar/ocultar
a nota.

## Passos
1. Abra as Configurações (engrenagem no editor, ou clique direito no ícone da barra de menus).
2. Vá na aba **Atalhos**.
3. Em **"Atalho personalizado ▸ Mostrar/ocultar a nota"**, clique no campo (**"Clique para gravar"**).
4. Pressione a combinação desejada — **precisa de pelo menos um modificador** (⌘, ⌥, ⌃ ou ⇧),
   ex.: **⌥⌘N**.
5. Pronto: o atalho passa a valer imediatamente, junto do ⌃⌥Espaço.

## Limpar / cancelar
- **Limpar**: clique em "Limpar" ao lado do campo, ou pressione **Delete** durante a gravação.
- **Cancelar** a gravação sem mudar nada: pressione **Esc**.

## Notas
- Vazio por padrão (nenhum atalho personalizado).
- Se a combinação já for usada pelo sistema/outro app, o registro pode falhar silenciosamente —
  escolha outra.
